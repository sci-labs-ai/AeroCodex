# Source and licensing checklist

LOCK-005 source/licensing gate for future formula promotion packets.

A promotion packet must show that each item below is satisfied before a validation status changes. This checklist does not promote status by itself.

- Source is public, citable, or otherwise legally usable for the repository and intended publication-supporting research workflow.
- Source is cited with a precise locator; do not copy source text wholesale.
- Proprietary standard text is not included.
- Large textbook derivations are not pasted into the repository.
- NASA/NACA public-domain sources are preferred where available, with title, report number, page/table/equation/section, and URL or archive locator.
- M07 is not treated as authoritative by itself.
- M07-derived candidates require independent public/citable/legal source confirmation before promotion.
- Each formula promotion packet records the exact source locator used for review.
- Each packet records reviewer, date, source identity, variables/units review, assumptions review, tests, and command evidence.
- Sidecars enrich metadata only; they cannot promote validation status.
- Promotion packets are required to change status and must be auditable.
- Do not claim NASA readiness, flight readiness, mission readiness, operational approval, certification, habitat safety, or life-support certification.
