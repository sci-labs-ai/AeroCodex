# AeroCodex public repository artifact manifest

This public repository contains the AeroCodex Rust workspace, governance metadata, validation/source-registry records, formula-vault metadata, nomenclature policy data, and research-only documentation.

## Public cleanup status

The public tree intentionally excludes:

- deployment helper scripts formerly kept under `scripts/`;
- prompt archives formerly kept under `archive/`;
- deployment prompt/operating-rule documents formerly kept under the deployment docs area;
- generated nomenclature inventory outputs and private terminology-pack tooling.

## Retained public artifact families

- Rust crates under `crates/` and the Rust `xtask` gate.
- Beta 1 concept CLI source and documentation.
- Validation cards, source-registry seeds, formula-vault contracts/manifests/resolutions, and equation inventory metadata.
- Thin-film BLSS research data, source-boundary records, and validation metadata.
- Nomenclature registries, schemas, templates, docs, and examples.
- Checksums for the current public file set in `checksums/SHA256SUMS`.

The cleanup does not change AeroCodex validation status. The repository remains research/preliminary-design software only and is not certified, flight-ready, mission-ready, operational, habitat-safe, medical, or regulated-use approved.
